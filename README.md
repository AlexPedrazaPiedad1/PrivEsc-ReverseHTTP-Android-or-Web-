# PrivEsc + ReverseHTTP (Android or Web)

Este repositorio reúne configuraciones, scripts y recursos orientados a la escalada de privilegios y al establecimiento de conexiones remotas mediante Reverse HTTP en entornos Android e iOS y aplicaciones web. Se centra en el uso de canales HTTP/HTTPS para la comunicación remota y local, facilitando la interacción con sistemas detrás de redes restringidas o con controles de firewall.

Incluye la preparación del entorno, gestión de sesiones remotas y adaptación a distintos escenarios, tanto en dispositivos Android como en entornos web. También aborda la integración de estos mecanismos con configuraciones específicas del sistema y servicios asociados.

El contenido está dirigido a usuarios con conocimientos en redes, sistemas móviles y entornos web que busquen trabajar con este tipo de configuraciones de forma organizada y práctica e educativa.
